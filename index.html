Backend Developer

A backend developer (aka "backender") is an IT specialist who writes server code and APIs, works with databases, and creates libraries. He is in the "shadow", as his work is difficult to notice immediately.

As a developer, it is important to have a solid understanding of what the internet is and how it works. It is the foundation upon which most modern software applications are built. In order to build effective, secure, and scalable applications and services, you need to have a solid understanding of how the internet works and how to leverage its power and connectivity. In this article we will cover the basics of internet including what it is how it works, some basic concepts, terminology and some common protocols that are used to build applications and services on the internet.

The Hypertext Transfer Protocol (HTTP) is the foundation of the World Wide Web, and is used to load webpages using hypertext links. HTTP is an application layer protocol designed to transfer information between networked devices and runs on top of other layers of the network protocol stack. A typical flow over HTTP involves a client machine making a request to a server, which then sends a response message.
